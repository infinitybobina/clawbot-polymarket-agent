# Автозапуск ClawBot (Windows Task Scheduler)

Запуск по расписанию каждый час.

**Важно:** этот файл (SCHEDULER.md) — только инструкция. Сообщения и ошибки автоматического запуска **не пишутся сюда**. Они записываются в файл **`clawbot_run.log`** в папке проекта. После каждого запуска (вручную или по расписанию) открывайте `clawbot_run.log` — там будет видно, загрузились ли рынки, сколько сигналов, ушло ли сообщение в Telegram или почему нет.

## Шаг 1: Открыть Планировщик заданий

- `Win + R` → `taskschd.msc` → Enter  
  или: Панель управления → Администрирование → Планировщик заданий

## Шаг 2: Создать задачу

1. Действия → **Создать задачу** (не «Простая задача»).
2. **Общие:** имя `ClawBot hourly`, выполнять для текущего пользователя.
3. **Триггеры** → Создать:
   - Начать задачу: **По расписанию**
   - Повторять каждый: **1 час**
   - Включено: да
4. **Действия** → Создать:
   - Действие: **Запуск программы**
   - Программа: полный путь к `python.exe`, например  
     `C:\Users\User\AppData\Local\Programs\Python\Python312\python.exe`  
     (узнать: в терминале `where python`)
   - Аргументы: полный путь к `main.py`, например  
     `c:\Dev\Clawbot\clawbot-polymarket-agent\main.py`
   - Рабочая папка: `c:\Dev\Clawbot\clawbot-polymarket-agent`
5. **Условия:** при желании снять «Запускать только при питании от электросети» для ноутбука.
6. ОК → ввести пароль пользователя, если попросит.

## Проверка

- В Планировщике: правый клик по задаче → **Выполнить**. В Telegram должен прийти отчёт.
- **Если задача завершается успешно, но в Telegram ничего не приходит:**
  1. В задаче во вкладке **Действия** обязательно укажите **Рабочую папку:** `c:\Dev\Clawbot\clawbot-polymarket-agent` (иначе скрипт не найдёт `.env`).
  2. В `main.py` теперь `.env` подхватывается из папки, где лежит `main.py`, а не из текущей папки — так сообщения уйдут в Telegram даже при другом «Рабочая папка». Всё равно лучше указать рабочую папку проекта.
  3. Проверьте `.env`: в нём должны быть строки `TELEGRAM_TOKEN=...` и `TELEGRAM_CHAT_ID=...` без лишних пробелов и комментариев на той же строке.
- **Логи автоматического запуска:** смотрите файл **`clawbot_run.log`** в папке проекта. Туда пишется каждый запуск (дата/время, Fetched N markets, Generated N signals, Telegram отправлен / не отправлен и т.д.). Если при автозапуске сигнал не пришёл в Telegram — откройте `clawbot_run.log` и найдите строку с WARNING или последний запуск по времени.

## Интервал запуска

Задача по умолчанию настроена на повтор **каждый 1 час**. Для ускорения тестирования можно поставить **10 минут** (в триггере: «Повторять каждые» → 10 минут). После завершения тестов верните 1 час или нужный интервал (например, каждые 4 часа).
